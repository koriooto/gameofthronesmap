# Известный мир — интерактивная карта

Интерактивная карта мира «Песни Льда и Пламени» / «Игры престолов»: Вестерос,
Эссос, Соториос, Летние острова, Иб, Ступени и дальние земли Востока.

Все описания локаций **без сюжетных спойлеров** — только география, устройство
и общеизвестная древняя история мира.

**Живая карта:** https://koriooto.github.io/gameofthronesmap/

## Возможности

- Запечённая рельефная подложка (hillshading, биомы, леса, реки, дюны,
  снега) поверх которой работает векторный интерактив; генератор подложки —
  в `gen/terrain.js` (открыть `/gen/terrain.html` через `npx vite`).
- Зум и панорамирование: колесо мыши, перетаскивание, двойной клик, кнопки.
- 154 локации: столицы, замки, города, городки, руины и примечательные места
  с карточками-описаниями, гербами и девизами правящих домов.
- Поиск по русским и английским названиям.
- Фильтры по типу локации и по 20 регионам трёх частей света; клик по региону
  Вестероса на карте подсвечивает его и фильтрует список.
- Плавный перелёт камеры к выбранной локации или региону.
- Режим «Путь героев» (включается отдельно, со спойлер-предупреждением):
  таймлайн по всем 73 сериям сериала и маршруты 8 персонажей на карте,
  с фишками героев и автопроигрыванием.

## Запуск

```bash
npm install
npm run dev      # режим разработки
npm run build    # сборка в dist/
npm run preview  # предпросмотр сборки
```

Стек: React 18 + Vite, без прочих зависимостей.

## Структура

```
src/
  data/regions.js     — регионы, континенты, типы локаций (полигоны Вестероса)
  data/locations.js   — база локаций с описаниями
  data/houses.js      — гербы и девизы домов
  map/shapes.js       — контуры Эссоса, островов, реки, горы, подписи морей
  map/markers.jsx     — значки типов локаций
  components/MapView.jsx      — SVG-карта, зум/пан, маркеры, регионы
  components/Sidebar.jsx      — поиск, фильтры, список локаций
  components/LocationCard.jsx — карточка выбранной локации
```

Фанатская работа по мотивам мира Джорджа Р. Р. Мартина. География стилизована
и не претендует на картографическую точность.
